# Linear Regression 의 가설과 비용

### Linear Hypothesis

* 예시
  * 일반적으로 학생이 공부를 많이 할수록 시험 성적이 향상
* 수식
  * 통상 1차원 그래프
    * y = ax + b
* 정확성 체크 = 비용 cost or 손실 loss
  * 가설과 실제값의 차이
  * 보통은 두 포인트간의 디스턴스를 단순히 minus 하지 않음
    * 결과가 양수, 음수로 나올 수 있기 때문에 제곱하는 것이 정확
  * 기본적인 cost function
    - n개 디스턴스 제곱의 합 / n = 디스턴스의 평균값
  * 결과값을 최소화 하는 다양한 알고리즘 활용
